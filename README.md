# 2D Object Management API

## Overzicht
Dit project is een backend API voor een Unity-applicatie.  
De API wordt gebruikt om 2D-omgevingen en objecten te beheren.

Gebruikers kunnen omgevingen en objecten aanmaken, bekijken en verwijderen.

Het project is opgebouwd met een vertical slice architectuur, waarbij elke functie (zoals Environment en Object2D) volledig los van elkaar is opgebouwd.


---

## Technologieën
- ASP.NET Core Web API
- Dapper
- Identity Core 
- HTTPS
- C#
